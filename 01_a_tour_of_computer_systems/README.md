# 1. A Tour of Computer System

## 1.1 Information Is Bits + Context

### Source Program

The source program written in text readable by humans is not as seen as it is. The information on the source code is bits + context, which allows us to understand machine representation. Here the word 'bits' matches computational representaiton, while the word 'context' make the representation understandable by humans.

> **My Opinion**
> The syntax on programming language is the rule for communication between human and computer, providing human contexts through the form of bits.
